##### 含义
* `NPM` 下载/安装/删除 管理。

##### 参数
> 不能为空，也不能有多个参数。

* `x.x.xx` - 任意已知版本的 `NPM` 。
* `latest` - 最新版本的 `NPM` 。
* `global` - 当前 `Node.js` 对应的 `NPM` 。

##### 子命令
* 不包含任何子命令。

##### 用法
```
gnvm npm x.xx.xx          :安装 x.xx.xx NPM
gnvm npm latest           :安装 latest  NPM
gnvm npm global           :安装 当前 Node.js 匹配的 NPM
```
