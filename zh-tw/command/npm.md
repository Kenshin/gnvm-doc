##### 含義
* `NPM` 下載/安裝/刪除 管理。

##### 參數
> 不能為空，也不能有多個參數。

* `x.x.xx` - 任意已知版本的 `NPM` 。
* `latest` - 最新版本的 `NPM` 。
* `global` - 當前 `Node.js` 對應的 `NPM` 。

##### 子命令
* 不包含任何子命令。

##### 用法
```
gnvm npm x.xx.xx          :安裝 x.xx.xx NPM
gnvm npm latest           :安裝 latest  NPM
gnvm npm global           :安裝 當前 Node.js 匹配的 NPM
```
